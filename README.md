## About

This repository contains the backend server side of the “Soil” application: an API built in JavaScript, using Node.js. It handles services such as connecting with Firebase, managing server behavior, etc.

---
## Features

- Basic server setup via server.js
- Firebase integration via firebase.js
- Package management via package.json / package-lock.json
- .gitignore to exclude build and secret files
- Ready for extension: create new endpoints, integrate database, authentication, etc.

## Getting Started

### Prerequisites
- Node.js (>= 14.x recommended) 
- npm or yarn

### Installation
```bash
git clone https://github.com/zvoosh/soil-backend.git  
cd soil-backend  
npm install
```
### Run Server
```bash
node server.js
```
